# MindmapMD — Feedback

This is the public issue tracker for **[MindmapMD](https://mindmapmd.app)**,
the proprietary VS Code mindmap editor that opens any `.md` file as an
interactive 2D mindmap, bidirectionally synced with the source.

This repo holds **only Issues** — no source code. The extension itself
ships from a private repository; this is where users report bugs and
request features in the open.

---

## File a bug

Found something broken? Please [open a Bug report](https://github.com/mindmapmd/feedback/issues/new?template=bug_report.yml).

A good report tells us:

- What you tried (the `.md` file shape, or the steps you took).
- What you expected vs. what happened.
- VS Code version and OS.
- A minimal `.md` snippet that reproduces it, if possible.

---

## Request a feature

Have an idea? [Open a Feature request](https://github.com/mindmapmd/feedback/issues/new?template=feature_request.yml).

We optimise hard for **fewer, sharper features**. The fastest way to get
a "yes" is to explain the underlying job-to-be-done, not just propose
a specific solution.

Things that intentionally won't ship:

- File format support beyond Markdown (we are Markdown-native, on purpose).
- Attachments other than images and PDF (zip, exe, docx, video are
  intentionally out — see Privacy Policy).
- AI features hosted by us (we lean on your existing Claude Code /
  Copilot / Cursor subscription instead).
- Realtime collaborative editing inside the extension (planned as a
  separate paid service, not in core).

## Install

VS Code Marketplace: **[mindmapmd.mindmapmd](https://marketplace.visualstudio.com/items?itemName=mindmapmd.mindmapmd)**.

Or from the command line:

```bash
code --install-extension mindmapmd.mindmapmd
```
